📌 Main Idea
CSS makes websites look beautiful and organized.
🎨 Key Points
Controls colors, fonts, spacing, layout Works with HTML elements Uses selectors to target elements
📦 Box Model
Every element has:
Content
Padding
Border
Margin
📐 Layout Tools
Flexbox → for row/column layout
Grid → for rows and columns layout
CSS = design + layout of the website
